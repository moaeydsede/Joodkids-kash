# VIP Cashbook (دفتر يومية الصندوق والمحافظ)

تطبيق ويب احترافي يعمل على GitHub Pages بدون سيرفر.
- شاشة دخول
- صفحة الأرصدة (مقبوضات/مدفوعات/رصيد لكل حساب)
- صفحة العمليات مع فلترة (حساب + من/إلى)
- صفحة الحسابات (إضافة/تعديل/حذف)
- صفحة المستخدمين (للـ Owner فقط) لإضافة/حذف مستخدمين من داخل النظام
- تصدير CSV وطباعة

## الدخول الافتراضي (المالك)
- Username: admin
- Password: 12345

## التشغيل محلياً
افتح `index.html` مباشرة.

## النشر على GitHub Pages
1) ارفع الملفات داخل Repo
2) من Settings > Pages
3) اختر Branch: main ثم /root
4) احفظ وسيظهر رابط الموقع

## ملاحظة أمنية
هذا مشروع Static؛ لا يوجد سيرفر.
كلمات المرور محفوظة محلياً داخل الجهاز (LocalStorage). لا تستخدم كلمات مرور حساسة.
